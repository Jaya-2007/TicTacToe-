# Tic Tac Toe (Java – Console)

A simple **console-based Tic Tac Toe game written in Java.
This project is designed for beginners to understand basic Java concepts like arrays, loops, conditions, and methods.

---

##  Features

* Two-player game (X vs O)
* 3×3 game board
* Checks for win and draw
* Simple and editable code
* Console-based (no GUI)

##  Technologies Used

* Java
* Scanner class (for user input)

##  How to Run

1. Clone the repository:
   bash
   git clone https://github.com/your-username/tic-tac-toe-java.git
  
2. Open the project in any Java IDE (VS Code / IntelliJ / Eclipse)
3. Compile and run:
4. bash
   javac TicTacToe.java
   java TicTacToe

##  How to Play

* Player X starts first
* Enter **row and column values (0–2)
* Players take turns
* First to align 3 symbols wins
* If the board fills with no winner → Draw
